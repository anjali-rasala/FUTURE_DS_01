# Business Sales Performance Analytics

## Objective
Analyze business Sales data to identify revenue trends, top-selling products, category performance, and regional insights using Power BI.

## Tools Used
- Power BI
- Superstore Dataset

## Dashboard Features
- Sales Trend Over Time
- Top Product by Sales
- Sales Contribution by Category
- Regional Sales Performance
- KPI Metrics
- Interactive Slicers

## Key Insights
- Technology category contributed the highest sales,
- West region generated the highest revenue,
- Sales showed overall upward growth over time,
- Top products significantly contributed to overall sales

## Recommendations
- Focus marketing efforts on high-performing categories,
- Improve strategies in lower-performing regions,
- Promote best-selling products further.
